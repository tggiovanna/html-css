Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado, usando a tag <delt>.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, usando a tag <ins>.

Texto sobrescrito

Para inserir coisas do tipo x20+3, usando a tag <sup>.

Texto subscrito

Para inserir coisas do tipo H20, usando a tag <sub>.
